How does it work?

  • The grid is filled with cells, each either alive or dead.
  • Each cell interacts with its eight neighbors (adjacent horizontally, vertically, or diagonally).
  • At each step, these rules apply:
    • Any live cell with <2 live neighbors dies (underpopulation).
    • Any live cell with 2 or 3 live neighbors lives on.
    • Any live cell with >3 live neighbors dies (overpopulation).
    • Any dead cell with exactly 3 live neighbors becomes alive (reproduction).
